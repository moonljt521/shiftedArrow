# 箭头消除

一个基于 `Vite + TypeScript` 的竖屏益智游戏。

## 玩法

点击箭头线段，让它沿着自身方向飞出屏幕。

如果头部前方的射线被阻挡，就不能消除，还会扣血。

## 核心规则

- 棋子是一条带方向的折线。
- 现在判定时，**自己身体也算阻挡**。
- 可以连续点击多个可消除棋子，不需要等上一条完全结束。
- 提示会标出当前可消除的棋子。
- 每关有倒计时、生命值和提示次数限制。
- 通关后会自动进入下一关，并保存解锁进度。

## 界面说明

- 左上角 `⚙`：暂停 / 设置菜单
- 左上角 `☰`：关卡选择
- 中间：当前关卡、生命值、倒计时
- 底部 `💡`：提示

## 项目结构

- `src/main.ts`：入口
- `src/gameController.ts`：游戏流程、点击、计时、通关
- `src/gridLogic.ts`：可消除判定、求解、难度评分
- `src/levels.ts`：关卡生成
- `src/renderer.ts`：SVG 渲染与动画
- `src/levelRules.json`：关卡参数与难度曲线

## 关卡生成思路

关卡不是手写固定图，而是按规则自动生成：

1. 先构造覆盖全盘的蛇形路径。
2. 再用 `backbite` 随机化制造大量弯折。
3. 再切分成多段箭头棋子。
4. 最后用求解器筛选可解关卡，并按目标难度挑选。

当前的关卡曲线是逐步增强的，后期会更接近高密度迷宫感。

## 设计说明

- 这是单机前端项目，进度保存在浏览器 `localStorage`
- 适配竖屏移动端布局
- 飞出动画是沿折线路径前移的，不是简单淡出
- 关卡生成和提示都基于同一套求解逻辑，避免提示失真

## 已知特点

- 默认共有 10 关
- 第 10 关会明显更接近高压迷宫风格
- 当前规则下，**自身也会阻挡自身的射线路径**
